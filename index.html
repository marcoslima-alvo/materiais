O que é Curva ABC?

A Curva ABC é uma técnica de classificação que separa itens em classes A, B e C conforme sua relevância econômica, normalmente usando o valor anual consumido (quantidade × preço). Itens da classe A concentram maior impacto financeiro; B são intermediários; C têm menor impacto, mas podem ser numerosos.

Como usar os modelos

  1. Preencha Item, Descrição, Qtde e Preço no arquivo Modelo.
  2. Calcule Valor Anual = Qtde × Preço e ordene por esse campo (maior para menor).
  3. Calcule % Participação de cada item sobre o total e, em seguida, o % Acumulado.
  4. Classifique como A até ~80%, B até ~95% e C para o restante.
